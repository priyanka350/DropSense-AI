# 📊 Dataset Exploration & Model Development

This file contains the complete machine learning development pipeline for **DropSense AI**, including dataset exploration, preprocessing, feature engineering, model comparison, explainability, and deployment preparation.

## Contents

- 📂 Dataset loading and exploration
- 📈 Exploratory Data Analysis (EDA)
- 🧹 Data preprocessing
- ⚙️ Feature engineering
- 🤖 Model comparison (Logistic Regression, Random Forest, XGBoost, LightGBM, CatBoost, Neural Network)
- 🏆 Final CatBoost model training
- 🔍 SHAP Explainable AI analysis
- 💾 Model export using Joblib
- 📄 Sample input template generation

## Dataset (data.csv)

- UCI Machine Learning Repository
- Predict Students' Dropout and Academic Success Dataset
- 4,424 student records

## Output

This pipeline generates:

- Trained CatBoost model (`catboost_dropout_model.pkl`)
- Feature names (`feature_names.pkl`)
- Sample input template (`sample_input_template.csv`)
